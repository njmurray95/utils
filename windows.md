# Command Prompt (v. 10.0)

Basic commands:

* cd -- change dir
* dir -- list all files (equivalent to `ls`)
* echo -- print text
* path -- show current system path

### History

Get history:
```
C:> doskey /history
```

### Path Manipulations

Show path:
```
C:> path
```

### Binary Compare

```
C:> fc /b <file 1> <file 2>
```
